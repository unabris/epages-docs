SSL Certificates renewal

Initial certificate creation

Motivation

  • SSL certificates lifetime is limited (90 days)
  • We want certificate renewal process to be automated

Challenges

  • Follow Let's Encrypt guidelines:
    • Honor rate limits
    • Spread interaction across the time
  • Work in a clustered environment

Certificate renewal highlights

  • Concern of ng-certificate-management only
  • Certificate renewal is a periodic activity
    • Renewal starts 30 days before expiration
    • In case of problems renewal is retried next day
  • Implemented using Quartz (supporting clustered mode)
